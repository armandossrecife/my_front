Sobre

MAI: uma aplicação para apoiar na tomada de decisão na gestão de almoxarifado.

Features Básicas

Um protótipo web que visa integrar as análises de dados e inferências (uso de Machine Learning e IA (LLM)) dos dados do almoxarifado.

Objetivo: criar uma aplicação de gestão inteligente de estoque, acompanhando tarefas rotineiras, analisando movimentações e compras de produtos. Utilizando técnicas de Machine Learning e linguagem de modelagem de grande escala (LLM), o MAI vai oferecer insights sobre o comportamento do estoque, permitindo a tomada de decisões estratégicas e eficientes. Através de um painel intuitivo, os usuários poderão acessar diversas funcionalidades, como a criação de dashboards personalizados, a análise movimentações, análises de compras por fornecedor e a execução de tarefas de forma otimizada.
